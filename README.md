# math

Реализация собственной версии математической библиотеки math.h.

## Introduction

В данном проекте Вам предстоит разработать свою версию стандартной библиотеки math.h на языке программирования Си. Эта библиотека реализует базовые математические операции, которые затем используются в различных алгоритмах.
